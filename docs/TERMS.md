## Description of service

TriageGuard provides **queue intelligence** for Reddit moderators: prioritized triage of reported or AutoModerator-filtered content, heuristic risk scoring, an explainability panel, and optional one-click approve/remove actions. TriageGuard **does not replace** Reddit’s native mod queue or moderator judgment.

## Eligibility

- The app requires **moderator** permissions on the subreddit where it is installed.
- The app owner must comply with [Reddit’s Developer Terms](https://www.redditinc.com/policies/developer-terms) and [Devvit Rules](https://developers.reddit.com/docs/devvit_rules).

## Moderator responsibilities

Moderators are solely responsible for:

- Final moderation decisions (approve, remove, ban, etc.)
- Configuring app installation settings (sensitivity, keywords, audit mode)
- Ensuring use complies with subreddit rules and Reddit Content Policy

TriageGuard suggestions and scores are **recommendations only**, not automated enforcement.

## Optional LLM enrichment

If enabled by the app owner, optional Groq-based enrichment may classify high-priority items. This feature:

- Requires an API key set by the app owner
- Sends limited content snippets to Groq’s API
- May be disabled; heuristics still function without it
